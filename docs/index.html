Introduction

Spinodal decomposition is the mechanism in which a single thermodynamic phase seprates into two without nucleation. This is often observed in binary alloy systems where a mixture of two metal species seperate into two coexisting phases. The time evoloution of a binary alloy system is commonly modelled using the Cahn-Hilliard equation, which is a non-linear parabolic partial differntial equation. This software solves the Cahn-Hilliard equation using either finite differnces and explicit time integration, or via spectral methods using fast fourier transforms. SpinDec2 also has the capability of alterning the definition of the chemical mobility of the system to be dependent on the tempearture or concentration fields or both. The definition for these mobility fields are described by the Darkens equation and Boltzmann statistics, and are outlined in Y. H. Wang et al., J. Appl. 12 85102 (2019)). SpinDec2 has also been parralleised using OpenMP and MPI to optimise the code to run with incresed efficincey and uses the netcdf package to write the output and checkpoint files.

Inputs

All input is given in the form 'key=value' 
        
Concentration_max=<real>  ==> upper bound of uniform distribution that initialises concentration field

Concentration_min=<real>  ==> lower bound of uniform distribution that initialises concentration field

Domain_x_size=<real>  ==> size of x space

Domain_y_size=<real>  ==> size of y space

Mobility_A=<real>  ==> Atomic mobility of species A (for temperature independent mobility)

Mobility_B=<real>  ==> Atomic mobility of species B (for temperature independent mobility)

free_energy_gradient_parameter=<real>  ==> strength of interfacial term (kappa)

Bulk_free_energy=<real>  ==>  bulk free energy scale factor (A) for case f(c) = Ac^2(1-c)^2

Checkpointing_interval=<intl>  ==> checkpointing interval

Checkpoint_output_file=<str>  ==> checkpoint file name

f(c)=<real, real, ...>  ==> list of polynomial coefficient values to set bulk free energy

Max_time=<real> ==> End time of simulation

dF_tolerance=<real> ==> Tolerence limit in change in total free energy

Random_Seed=<real> ==> Seed to reproduce result

Use_input=<real> ==>  overwrites checkpoint meta data when set to one

Excitation_A=<real> ==>  Exciation energy of species A (for temperature dependent mobility)

Excitation_B=<real> ==>  Exciation energy of species B (for temperature dependent mobility)

temperature_max=<real>  ==> upper bound of uniform distribution that initialises temperature field

temperature_min=<real>  ==> lower bound of uniform distribution that initialises temperature field

Problem=<strl>  ==> defines which mobility definition to use ('Consatnt', 'NonTemp', 'Temp')

Stabilization_Term=<int>  ==> controls the stability of spectral method solver
