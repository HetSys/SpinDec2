SpinDec2
PX915 group A software development project for HetSys in modelling spinodal decomposition
Input file

Inputs

Inputs are given in the form form 'key=value'

  • Concentration_max=<real> ==> upper bound of uniform distribution that initialises concentration field.
  • Concentration_min=<real> ==> lower bound of uniform distribution that initialises concentration field.
  • Domain_x_size=<int> ==> size of x space.
  • Domain_y_size=<int> ==> size of y space.
  • Mobility_A=<real> ==> Atomic mobility of species A \(M_A\), (for constant and temperature independent mobility).
  • Mobility_B=<real> ==> Atomic mobility of species B \(M_B\), (for constant and temperature independent mobility).
  • free_energy_gradient_parameter=<real> ==> strength of inter-facial term \(\kappa\).
  • Bulk_free_energy=<real> ==> bulk free energy scale factor (\(A\)) for case \(f(c) = Ac^2(1-c)^2\).
  • Checkpointing_interval=<int> ==> check-pointing interval.
  • Checkpoint_output_file=<str> ==> check-point file name.
  • Checkpoint_input_file=<str> ==> input check-point file (optional).
  • f(c)=<real, real, ...> ==> list of polynomial coefficient values to set bulk free energy.
  • Max_time=<real> ==> End time of simulation.
  • dF_tolerance=<real> ==> Tolerance limit in change in total free energy.
  • Random_Seed=<int> ==> Seed to reproduce result.
  • Use_input=<int> ==> overwrites checkpoint meta data when set to one.
  • Excitation_A=<real> ==> Excitation energy of species A \(E_A\), (for temperature dependent mobility).
  • Excitation_B=<real> ==> Excitation energy of species B \(E_B\), (for temperature dependent mobility).
  • temperature_max=<real> ==> upper bound of uniform distribution that initialises temperature field.
  • temperature_min=<real> ==> lower bound of uniform distribution that initialises temperature field.
  • Problem=<strl> ==> defines which mobility definition to use ('Constant', 'NonTemp', 'Temp') and for spectral methods ='spectral'.
  • Stabilization_Term=<int> ==> controls the stability of spectral method solver (must be negative).
