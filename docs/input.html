Inputs

In SpinDec2, the user can specify the values many of the physical varibales when modelling spindoal decomposition as well as controlling the checkpointing and version of the code. This must be done via an input.txt that has to be in the same directory as your spindec executable.

A list of all the user inputs are outlined below with there respective names and types

  • concentration_max = value (real) \( \rightarrow \) Upper bound of uniform distribution that initialises concentration field. Thus \(c(:,:,1) = \mathcal{U}(\text{Concentration_min}, \text{Concentration_max})\).


  • concentration_min = value (real) \( \rightarrow \) Lower bound of uniform distribution that initialises concentration field. Thus \(c(:,:,1) = \mathcal{U}(\text{Concentration_min}, \text{Concentration_max})\).


  • domain_x_size = value (int) \( \rightarrow \) Number of grid cells in the x dimesnion \(N_x\).


  • domain_y_size = value (int) \( \rightarrow \) Number of grid cells in the y dimesnion \(N_y\).


  • mobility_A = value (real) \( \rightarrow \) Atomic mobility of species A, \(M_A\), (for constant and temperature independent mobility).


  • mobility_B = value (real) \( \rightarrow \) Atomic mobility of species B, \(M_B\), (for constant and temperature independent mobility).


  • free_energy_gradient_parameter = value (real) \( \rightarrow \) Strength of inter-facial term \(\kappa\).


  • bulk_free_energy = value (real) \( \rightarrow \) Bulk free energy scale factor (\(A\)) for case \(f(c) = Ac^2(1-c)^2\).


  • checkpoint_interval = value (int) \( \rightarrow \) Check-pointing interval.


  • checkpoint_output_file = "value" (str) \( \rightarrow \) Check-point output file name.


  • checkpoint_input_file = "value" (str) \( \rightarrow \) Input check-point file (optional)


  • f(c) = value, value, value, ... (int, real, real, real, ...) \( \rightarrow \) List of polynomial coefficient values to set bulk free energy. The first value must be an integer which specifies the number of coefficients.


  • max_time = value (real) \( \rightarrow \) End time of simulation.


  • random_seed = value (int) \( \rightarrow \) Seed to reproduce random grid initialised from previous run.


  • use_input = value (int) \( \rightarrow \) Overwrites checkpoint meta data when set to one.


  • excitation_A = value (real) \( \rightarrow \) Excitation energy of species A, \(E_A\), (for temperature dependent mobility).


  • excitation_B = value (real) \( \rightarrow \) Excitation energy of species B, \(E_B\), (for temperature dependent mobility).


  • temperature_max = value (real) \( \rightarrow \) Upper bound of uniform distribution that initialises temperature field. Thus \(T = \mathcal{U}(\text{temperature_min}, \text{temperature_max})\).


  • temperature_min = value (real) \( \rightarrow \) Lower bound of uniform distribution that initialises temperature field. Thus \(T = \mathcal{U}(\text{temperature_min}, \text{temperature_max})\)


  • problem = "value" (str) \( \rightarrow \) Defines which mobility definition to use either "constant";, "nontemp";, "temp", and for spectral methods "spectral"


  • stabilization_term = value (int) \( \rightarrow \) Controls the stability of spectral method solver (must be negative).


  • write_frequency = value (int) \( \rightarrow \) Controls the rate at which frames are written. If set to 1, then every frame is written, if 10 then every ten frames are written etc. If set to 0 then only the last frame is written.


  • single_precision = value (int) \( \rightarrow \) Controls if data is written in single precision (value = 1) or not (value = 0). Writing in single precision will speed up the writing process.


An example of input.txt that can be used in SpinDec2 is shown below:

  
  
concentration_max = 0.9
concentration_min = 0.1
domain_x_size = 60
domain_y_size = 60
mobility_a = 4
mobility_b = 4
free_energy_gradient_parameter = 0.0001
bulk_free_energy = 1
f(c) = 5,0,0,1,-2,1
checkpoint_interval = 5000
checkpoint_output_file = "Out.cp"
checkpoint_input_file = "Out.cp"
max_time = 1e-6
time_step = 1e-10
random_seed = -1
use_input = 0
excitation_a = 10000
excitation_b = 20000
temperature_min = 8
temperature_max = 10
problem = "constant"
stabilization_term = -1
write_frequency = 1000
single_precision = 1
